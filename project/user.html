权限控制-页面访问权限 #

实现：需要登录的页面，需要判断是否有token

步骤：

  1. 定义白名单
  2. 判断，如果没有登录且不在白名单内，跳转到登录

代码

ts
// 访问权限控制
router.beforeEach((to) => {
  // 1. 定义白名单
  const whiteList = ["/login"];
  // 用户仓库
  const store = useUserStore();

  // 2. 如果没有登录且不在白名单内，去登录
  if (!store.userInfo?.token && !whiteList.includes(to.path)) return "/login";
  // 否则不做任何处理
});
