vuex速记 ​

本质： 全局状态（数据）管理工具

作用： 可以实现跨组件通信

五大核心模块：

  • state：数据
  • Mutation：修改数据（同步）
  • Actions：处理异步行为
  • Getters：计算属性
  • Modules：数据分模块维护

五大核心模块 ​

State ​

作用：声明数据

语法：

js
// 声明数据
state: {  
  // 属性名就是数据变量名，类似data函数
  xxx: 值,
  zzz: 值,
}


// 组件内，获取数据
computed: {
	...mapState(['xxx'， 'zzz'])
}

  1. 准备actions 和 mutations
jsx
import axios from 'axios'

export default {
  namespaced: true,
  state () {
    return {
      list: []
    }
  },
  mutations: {
    updateList (state, payload) {
      state.list = payload
    }
  },
  actions: {
    async getList (ctx) {
      const res = await axios.get('http://localhost:3000/cart')
      ctx.commit('updateList', res.data)
    }
  }
}

底部总价展示 ​

  1. 提供getters
jsx
getters: {
  total(state) {
    return state.list.reduce((p, c) => p + c.count, 0);
  },
  totalPrice (state) {
    return state.list.reduce((p, c) => p + c.count * c.price, 0);
  },
},
